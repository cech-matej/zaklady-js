Doplň vhodné odpovědi a využij k tomu nečíslovaného seznamu.

  • Navzdory slovu Java v názvu není totožný s jazykem Java, ale používá podobnou syntax příkazů

2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. JavaScript is case sensitive
  2. Statements should end in a semicolon (;)
  3. Strings have to be enclosed in quotation marks
  4. To enter comments in the script, use "//" to start a single line comment or the combination of "/*" and "*/" to enclose a multi-line comment
  5. To increment a variable, such as a = a + 1, you can use a++. You can decrement a variable in the same way, as in a--

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* Objeví se okno, které říká 'Nezadal jsi správné heslo!' */
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
let password = prompt('Heslo:');

confirm()

/* Při potvrzení (stiskem tlačítka OK), stránka přesměruje uživatele na adresu 'https://seznam.cz' */
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování: Na méně důvěřivých stránkách se můžou pop-up okna zneužívat. Pop-up okna mohou uživatele přesměrovat na jinačí stránku bez toho aniž by si to uživatel přál.
